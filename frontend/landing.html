The Problem

Multi-agent systems break
in ways you can't see.

As agent architectures grow more complex, failures become invisible. Communication degrades silently, handoffs lose context, and goals drift without warning.

Agents fail silently

Multi-agent systems break without clear error signals. An agent can return plausible but wrong output, and the swarm continues as if nothing happened.

No visibility into handoffs

Parent-child agent communication is opaque. When one agent passes context to another, you can't see if the message was understood or mangled in translation.

Drift goes undetected

Agents slowly deviate from the user's original goal. Without periodic checkpoints, you only discover the drift when the final output is completely wrong.

Capabilities

Forensic-grade analysis
for every agent execution.

Four core modules that give you complete visibility into how your agents communicate, execute, and deliver results.

DAG Reconstruction

Automatically build execution graphs from raw agent logs. Supports linear chains, fan-out, fan-in, and diamond architectures. Visualize the full topology of any multi-agent trace.

Core

Edge Similarity Analysis

Detect communication drift between agents using cosine similarity scoring. Every parent-child edge is measured and flagged when input-output alignment drops below threshold.

Task Progress Monitoring

LLM-verified checkpoints ensure agents stay aligned with the user's original goal. Periodic analysis detects when execution drifts before it causes downstream failures.

Prompt Optimization

Iteratively improve agent prompts against gold-standard outputs. Measure the effect of prompt changes on downstream agent performance across the entire DAG.

How It Works

Three steps to full visibility.

From raw agent logs to actionable insights in under a minute. No SDK required.

Drop in your agent execution logs in JSON or Excel format. ArcanaAI ingests traces from any multi-agent framework.

pipeline.py 
from arcana_ai import AgentDAG, EmbeddingMatchAnalyzer

# Build DAGs from your agent execution logs
dags = AgentDAG.from_agent_logs(your_logs)

# Analyze communication quality on each edge
for trace_id, dag in dags.items():
    analyzer = EmbeddingMatchAnalyzer(dag, threshold=0.8)
    results = analyzer.analyze()

    # Flag edges with communication drift
    red_edges = [r for r in results if r["status"] == "RED"]
    print(f"{trace_id}: {len(red_edges)} issues found")
